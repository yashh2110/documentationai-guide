## Initial Setup

You begin your Paperguide journey by creating an account and configuring your workspace. This process takes under `5` minutes and sets the foundation for organized documentation. Follow the guided tour to import existing files if needed.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit paperguide.com and enter your details. Verify your email to activate.
  </Step>
  <Step title="Create Workspace" icon="home">
    Name your space, like `Team Docs`, and select a template for quick start.
    ```javascript
    // Optional: Configure initial settings via API
    const config = {
      name: 'Team Docs',
      template: 'basic'
    };
    await fetch('/api/workspaces', { method: 'POST', body: JSON.stringify(config) });
    ```
  </Step>
  <Step title="Invite Members" icon="users">
    Add team emails and assign roles: admin, editor, or viewer.
  </Step>
</Steps>

<Callout kind="tip">
  Use the dashboard to monitor activity and set notifications for changes.
</Callout>
