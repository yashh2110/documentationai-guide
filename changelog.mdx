## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration editing for multiple users.
- Introduced custom themes for workspace branding.

## Improvements

- Enhanced search with fuzzy matching for better results.
- Optimized performance for large documents (`>500` pages).

## Bug Fixes

- Fixed import issues with certain Markdown variants.
- Resolved notification delays in team invites.

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["breaking", "feature"]}>

## Breaking Changes

- Updated API endpoints; migrate to new `/v2/documents` paths.

## New Features

- Integrated GitHub sync for automatic doc updates from repos.
- Added analytics dashboard for usage insights.

## Improvements

- Improved mobile responsiveness for on-the-go editing.

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Support for MDX components in the editor.
- Folder-level permissions for granular access.

## Bug Fixes

- Corrected tagging system overlaps.
- Fixed export to PDF formatting errors.

</Update>
